# Azure AI Operations

> This repository contains the course content and architecture guidance for operating machine learning and generative AI workloads on Azure.

## Scope

The learning site combines practical operating guidance from these source repositories:

- [Azure-MLOps-Overview](https://github.com/Cloud2BR-MSFTLearningHub/Azure-MLOps-Overview)
- [GenAIOpsMaturityLevels](https://github.com/Cloud2BR-MSFTLearningHub/GenAIOpsMaturityLevels)

It explains where MLOps and GenAIOps share practices, where their operational concerns differ, and how teams can evolve from exploratory work to governed production services.

## Structure

- `docs/01-foundations/`: shared operating model, roles, and lifecycle concepts
- `docs/02-mlops/`: Azure ML maturity and production model lifecycle
- `docs/03-genaiops/`: LLM application maturity, evaluation, and monitoring
- `docs/04-enterprise/`: delivery, FinOps, governance, and security
